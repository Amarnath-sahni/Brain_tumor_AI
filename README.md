# Brain Tumor Detection System 🧠

This project is a full-stack Brain Tumor Detection System built using the MERN stack (MongoDB, Express.js, React.js, Node.js). It focuses on assisting early detection of brain tumors using MRI scan data while ensuring a secure, user-friendly, and accessible experience.

The application includes a two-step authentication (2FA) system to protect user data and maintain privacy. The frontend is developed with React.js, providing a responsive and intuitive interface for smooth user interaction.

A key feature of the system is its test case-based validation approach, which helps reduce unnecessary MRI scans and associated costs by improving decision-making efficiency.

In addition to detection functionality, the platform offers a patient-focused blogging system where users can share their experiences and recovery journeys, creating a supportive community.

## 🚀 Tech Stack
- Frontend: React.js, JavaScript  
- Backend: Node.js, Express.js  
- Database: MongoDB  
- Authentication: Two-Factor Authentication (2FA)  

## 🎯 Objective
To build a scalable and secure system that improves early diagnosis, reduces healthcare costs, and supports patient interaction.
